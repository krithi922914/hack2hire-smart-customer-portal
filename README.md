# AI Customer Retention Suite

Full-stack customer retention dashboard with MongoDB. Data persists after refresh ✅

## Features
- MongoDB database (real persistence)
- Add/delete customers
- Risk scoring (High/Medium/Low)
- AI analyst chat
- CSV export
